## 一个基于parcel打包工具的react脚手架


#### 21:46 星期三 2019年7月31日 添加react-router-dom支持

### 关键词

`react`、`parcel` 、`react-router-dom` 、`react-redux`

## 使用方法
```
git clone https://github.com/qiuquanwu/pareact
```
```
npm i
```
```
npm run dev
```
打开浏览器，输入http://localhost:1234/
